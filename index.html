Gemini API ключ

Для генерації потрібен безкоштовний ключ Google Gemini

1. Зайди на aistudio.google.com/apikey

2. Натисни «Create API key» → скопіюй

3. Встав ключ нижче — збережеться тільки у твоєму браузері

🔒 Ключ зберігається лише у твоєму браузері. Реєстрація на Google безкоштовна.
