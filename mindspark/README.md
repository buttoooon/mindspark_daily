# ✨ MindSpark Daily

Перевірені наукові факти та мудрість великих мислителів — щодня.

## 🚀 Як задеплоїти на GitHub Pages

1. Створи новий репозиторій на [github.com](https://github.com/new)
2. Назви його, наприклад, `mindspark-daily`
3. Завантаж всі три файли (`index.html`, `style.css`, `app.js`) у репозиторій
4. Перейди в **Settings → Pages**
5. У розділі **Source** вибери гілку `main` і натисни **Save**
6. Через ~1 хвилину сайт буде доступний за адресою:
   `https://твій-нік.github.io/mindspark-daily`

## 📁 Структура

```
mindspark/
├── index.html   ← головна сторінка
├── style.css    ← всі стилі
├── app.js       ← вся логіка + Claude API
└── README.md
```

## ✅ Функціонал

- 🌐 Перемикач мови УК / EN
- 🤖 Генерація перевірених фактів через Claude AI з веб-пошуком
- 🔗 Реальні джерела (Wikipedia, DOI, Stanford тощо)
- 🔖 Збереження і вподобання карток
- 👤 Реєстрація / Вхід / Відновлення паролю
- 📊 Профіль зі статистикою
- 💾 Все зберігається в localStorage браузера
